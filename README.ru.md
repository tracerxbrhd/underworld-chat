# Underworld Chat

Underworld Chat — это заготовка приватного мессенджера, построенного вокруг анонимных аккаунтов, свободной регистрации и сессий устройств вместо телефонных номеров.

## Форма продукта

- Публичный лендинг для гостей
- SPA-интерфейс диалогов после авторизации
- Список диалогов слева и активный чат справа
- Свободная регистрация без обязательных invite-кодов
- Редактируемый профиль: аватар, описание, дата рождения, отображаемое имя
- Неизменяемый логин аккаунта (`public_id`)
- Заготовка личного канала заметок с тем же поведением, что и у обычного чата
- Поиск по чатам и пользователям с быстрым созданием нового 1:1 диалога

## Стек

- Backend: Django + Django REST Framework + ASGI/Channels
- Realtime: WebSocket over ASGI
- БД: PostgreSQL
- Кэш / presence / broker: Redis
- Frontend: React + Vite + TypeScript
- Файлы: MinIO (S3-совместимое хранилище)
- Фоновые задачи: Celery
- Reverse proxy: Nginx
- Локальная разработка: Docker Compose

## Мультиязычность

Репозиторий подготовлен под английский и русский:

- двуязычная документация репозитория;
- словарь переводов и переключатель языка на фронтенде;
- настройки Django с включенными `en` и `ru`;
- задел под дальнейшие gettext-переводы на backend.

## Лицензия

Проект распространяется под Apache License 2.0 с файлом `NOTICE`, который сохраняет атрибуцию автору `tracerxbrhd`.

## Локальный старт

1. Скопировать `.env.example` в `.env`.
2. Проверить секреты и порты.
3. Запустить стек через `docker compose up --build`.
4. Открыть `http://localhost`.

## Деплой

- Production compose: `docker-compose.prod.yml`
- Руководство для сервера: `docs/deployment.ru.md`
- Быстрый bootstrap Ubuntu 24: `scripts/bootstrap-ubuntu-24.sh`
- Полный снос production-стека: `scripts/destroy-prod.sh`
